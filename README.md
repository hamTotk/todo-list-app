# TODOリスト - タスク管理アプリ

ブラウザベースのTODOリスト管理アプリケーションです。無制限の階層的サブタスク、柔軟な繰り返しタスク、タグ管理などの機能を備えています。
制作者のメモ：こちらはAIエージェントを試しに使ってみるために作った完全に自分用のアプリです。

## 技術スタック

- **フロントエンド**: バニラ JavaScript (ES6+), HTML5, CSS3
- **データ永続化**: LocalStorage
- **外部依存**: なし（フレームワーク不使用）

## 機能一覧

### 基本機能
- タスクの作成・編集・削除
- 優先度設定（高・中・低）
- 期限設定と期限状態の表示（期限切れ、今日、明日）
- 完了/未完了の切り替え

### グループ管理
- 複数のタスクグループを作成・管理
- グループ間のタスク分類
- 「すべて表示」で全グループ横断表示

### サブタスク機能
- 無制限の階層的サブタスク
- 展開/折りたたみ表示
- 親タスク完了時の一括完了オプション
- サブタスク進捗表示（例: 2/5）

### 繰り返しタスク
- 繰り返しパターン: 毎日、毎週、毎月、カスタム間隔、曜日指定
- 終了条件: 無期限、回数指定、日付指定
- 完了時の挙動: 即座に次回生成、期限時に生成、リセット

### タグ機能
- 複数タグの割り当て
- タグフィルタリング（OR条件）
- タグ管理（追加・編集・削除）

### 統計情報
- 基本統計: 全体数、未完了数、完了数
- 詳細統計: 期限別、優先度別、タスクタイプ別

### その他
- ダークモード/ライトモード切り替え
- レスポンシブデザイン（モバイル対応）
- 完了時のアニメーションエフェクト

## 使い方

### 基本操作

1. **タスクの追加**: 右上の「+ 新しいタスク」ボタンをクリック
2. **タスクの編集**: タスクの編集ボタン（鉛筆アイコン）をクリック
3. **タスクの削除**: タスクの削除ボタン（ゴミ箱アイコン）をクリック
4. **完了切り替え**: チェックボックスをクリック

### サブタスク

1. 親タスクの「+」ボタンをクリックしてサブタスクを追加
2. 「▼」ボタンで展開/折りたたみ
3. 親タスクを完了する際、未完了のサブタスクがあれば確認ダイアログが表示されます

### 繰り返しタスク

1. タスク作成/編集時に「繰り返し設定」をチェック
2. 繰り返しタイプを選択（毎日、毎週など）
3. 終了条件を設定（任意）
4. 完了時の挙動を選択
   - **即座に次回生成**: 完了すると次のタスクがすぐに作成される
   - **期限時に生成**: 次の期限日になったらタスクが生成される
   - **リセット**: 完了状態をリセットして同じタスクを再利用

## 起動方法

ローカルサーバーを起動してブラウザでアクセスします。

```bash
# Python 3の場合
python -m http.server 8000

# その後ブラウザで http://localhost:8000 を開く
```
